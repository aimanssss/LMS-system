A modern learning management system for WordPress.

== Description ==
BinawebKu LMS provides tools to create and manage online courses.

== Installation ==
1. Upload the plugin files to `/wp-content/plugins/binawebku-lms`.
2. Activate the plugin through the 'Plugins' screen.
